Tentang Saya

Halo, saya seorang pengembang web dan mahasiswa Informatika di Universitas Pamulang. Saya memiliki ketertarikan di bidang frontend development, khususnya dalam CSS, JavaScript, dan pengembangan antarmuka pengguna yang responsif.

Proyek Saya

Proyek 1: Pembuatan Portfolio Pribadi

Dalam proyek Tugas 1 ini, saya telah menyelesaikan pembuatan portfolio pribadi yang bertujuan untuk menampilkan keterampilan, pengalaman, dan pencapaian saya dalam bidang Teknik Informatika

Proyek 2: Pembuatan Form-Login

Dalam proyek Tugas 2 ini, saya telah menyelesaikan pembuatan form login yang bertujuan untuk memberikan pengalaman pengguna yang aman dan efisien dalam mengakses aplikasi.

Proyek 3: Pembuatan Kalkulator

Dalam proyek Tugas 3 ini, saya telah menyelesaikan pembuatan aplikasi kalkulator yang bertujuan untuk membantu pengguna dalam melakukan perhitungan matematika dasar dengan mudah dan cepat. Proyek ini dirancang untuk memperdalam pemahaman saya tentang logika pemrograman, algoritma, serta penggunaan HTML, CSS, dan JavaScript dalam pengembangan aplikasi web.

Proyek 4: Pembuatan Dashboard

Dalam proyek Tugas 4 ini, saya telah menyelesaikan pembuatan dashboard yang bertujuan untuk memberikan visualisasi data yang informatif dan interaktif. Proyek ini dirancang untuk membantu pengguna dalam menganalisis dan memahami data dengan lebih baik, serta untuk meningkatkan kemampuan saya dalam pengembangan aplikasi berbasis web menggunakan teknologi terkini.
